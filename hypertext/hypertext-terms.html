Hypertext Terms

Anchor
An area within a node which is the source or destination of a link. The anchor may be the whole of the node content.
Annotation
Additional notes or commentary on a document or piece of content. Annotation and multiuser collaboration
Browser
The software used to access and navigate web content.
Button
An interactive element used to perform an action in a user interface.
Card
A container for grouping related content.
Client
Software that accesses services provided by a server.
Daemon
A background process that performs tasks such as building indexes and back-links. Placeholder links: building back-links, building indexes, back-links.
Database
A structured collection of data that can be accessed and manipulated.
Document
A discrete unit of information that can be displayed, edited, or linked.
Domain
A specific area or category in a network or system.
External
Referring to resources or links outside the current site.
Host
A computer or server providing services or resources to clients.
Hypermedia
Media that uses hyperlinks to connect text, images, video, and audio. Hypermedia overview
Internal
Referring to resources or links within the same site or document.
Link
A relationship between two anchors, stored in the same or different database.
Navigation
Techniques and tools for moving through content. Navigation techniques and tools
Node
A unit of information, also known as a frame.
Path
The route or location of a file or resource in a system.
Reader
An individual or software consuming the content.
Server
A computer or software that provides resources or services to clients.
Tracing
The process of following links or document paths to understand connections.
Web
The global system of interconnected documents and resources linked by hyperlinks. WWW
